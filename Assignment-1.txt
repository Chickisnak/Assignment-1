1.Explain the use of JavaScript ( or What you can do using a JavaScript)

 Answer-
        JavaScript is a text-based programming language used both on the client-side and server-side 
        that allows you to make web pages interactive.Or In other Words, we can say that JavaScript insert life to Website.

2.What is the difference between client-side and server-side?
  Answer-
         Client-Side: Everything in a web application that is displayed or takes place on the client(end user device).
         Server-Side: Server-side means that the processing takes place on a web server.
3.What is Nodejs?
  Answer-
         Node JS is an interpreter or running environment for a JavaScript programming language that holds many excesses, 
         it requires libraries that can easily be accessed from JavaScript programming for better use.

4.Explain Scope in JavaScript?
  Answer-
         Scope refers to the availability of variables and functions in certain parts of the code.

         In JavaScript, a variable has two types of scope:

         1.Global Scope
         2.Local Scope

7.Explain DOM in your own word.
  Answer-
         The Document Object Model (DOM) is a programming interface for web documents. 
         It represents the page so that programs can change the document structure, style, and content. 
         The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.
